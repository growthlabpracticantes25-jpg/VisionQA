VisionQA
Descripción del proyecto
VisionQA es un sistema de control de calidad asistido por inteligencia artificial orientado al área de manufactura.
Actualmente el sistema permite capturar imágenes en tiempo real utilizando Python y OpenCV mediante una cámara web, clasificando las imágenes en categorías de piezas buenas y malas para la generación de un dataset inicial.
Objetivo
Desarrollar un prototipo funcional capaz de capturar y organizar imágenes para futuras etapas de entrenamiento de modelos de inteligencia artificial aplicados al control de calidad.
Tecnologías utilizadas
Python
OpenCV
Visual Studio Code
Funcionalidades actuales
Captura de video en tiempo real.
Guardado automático de imágenes.
Clasificación manual de imágenes buenas y malas.
Organización automática del dataset.
Cómo ejecutar el sistema
Abrir terminal en Visual Studio Code.
Ejecutar el siguiente comando:
py visionqa.py 
Controles del sistema
b→ Guardar imagen buena.
m → Guardar imagen mala.
q → Cerrar el sistema.
Estado actual del proyecto
El proyecto se encuentra en Sprint 2, enfocado en la construcción del sistema de captura y generación del dataset inicial para futuras etapas de entrenamiento con inteligencia artificial.
